## Foto's toevoegen aan activiteiten
1. Plaats foto in `assets/img/activities/`
2. Open `assets/js/activiteiten.js`
3. Vul bij een activiteit `foto` in, bv:
   `foto: "assets/img/activities/cegeka.jpg"`

## GitHub Pages
- Settings -> Pages
- Deploy from branch: `main`
- Folder: `/ (root)`
